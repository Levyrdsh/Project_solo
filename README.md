# CodePulse – Sistema de Cálculos em C

Descrição do Projeto


O CodePulse é um sistema de cálculos desenvolvido em C, criado para facilitar o dia a dia de pessoas que precisam realizar diferentes tipos de cálculos de forma simples e rápida.
O projeto integra várias funcionalidades em um único programa, oferecendo praticidade, acessibilidade e tecnologia para todos os perfis de usuários.

---> Alguns exemplos de suas funcionalidades:

Equações de 1º grau

Equações de 2º grau

Conversor de temperatura (°C, °F, K)

Cálculo de IMC

Interface via terminal

Fácil expansão para novos cálculos


---> Cenário Atual / Problema Encontrado


O Brasil forma cerca de 53 mil profissionais de TI por ano, mas o mercado demanda em média 159 mil, o que significa que apenas 33% da necessidade é suprida.

Além disso, pequenos comércios e comunidades ainda têm pouco acesso à tecnologia e ferramentas digitais básicas.

---> Solução Proposta

Levar tecnologia simplificada e acessível a diferentes públicos.

Oferecer cursos básicos de tecnologia para pessoas com dificuldade no manuseio digital.

Criar sistemas práticos que ajudem pequenos negócios a se informatizarem.
